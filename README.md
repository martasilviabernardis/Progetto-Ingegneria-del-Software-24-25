# Progetto-Ingegneria-del-Software-24-25
Progetto software in Java che implementa il gioco da tavolo Galaxy Trucker secondo il regolamento ufficiale. Architettura client-server con RMI, interfaccia CLI, paradigma MVC+. Il sistema gestisce costruzione delle navi, carte avventura, turni di gioco e sincronizzazione dello stato tra i giocatori.
